Das Grundgerüst einer Homepage

Navigation

Element Funktion
<html> Zeigt dem Interpreter den Start der Html Seite an
<head> Meta informationen die dem Interpreter helfen
<meta> Eine Information die dem Interpreter sagen was zu erwarten ist (z.b. charset=utf-8)
<style> CSS Styles werden hier angegeben
<body> Ab hier beginnt die Homepage
<head> Die Überschrift der Seite
<br> Einfach eine leere Zeile
<table> Der Anfang der Tabelle
<thead> Die erste Zeile/Überschrift der Zellen
<tr> Eine Tabellenzeile (Tablerow)
<th> Eine TabellenÜberschrift (Tableheader) zelle
